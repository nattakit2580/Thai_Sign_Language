# Thai_Sign_Language

ภาษามือไทย ผู้ที่บกพร่องด้านการได้ยินต้องการที่จะสื่อสารกับบุคคลทั่วไปให้เข้าใจ ปัญหาที่พบเจอบ่อย คือการสื่อสารกับบุคคลทั่วไป การนำเสนอระบบตรวจจับท่าทางภาษามือไทย ผ่านทางเว็บไซต์เพื่อใช้งาน ทำให้คนที่ไม่รู้ภาษามือได้รับประโยชน์ว่าผู้ที่พิการนั้นต้องการที่จะสื่อสารอะไร การจัดเก็บข้อมูลภาษามือเบื้องต้น กับ 10 สัญญาณมือ ในยามคับขัน
