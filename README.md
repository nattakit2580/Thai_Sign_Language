# Thai_Sign_Language

ภาษามือไทย ผู้ที่บกพร่องด้านการได้ยินต้องการที่จะสื่อสารกับบุคคลทั่วไปให้เข้าใจ ปัญหาที่พบเจอบ่อย คือการสื่อสารกับบุคคลทั่วไป การนำเสนอระบบตรวจจับท่าทางภาษามือไทย ผ่านทางเว็บไซต์เพื่อใช้งาน ทำให้คนที่ไม่รู้ภาษามือได้รับประโยชน์ว่าผู้ที่พิการนั้นต้องการที่จะสื่อสารอะไร การจัดเก็บข้อมูลภาษามือเบื้องต้น กับ 10 สัญญาณมือ ในยามคับขัน

โปรเจกต์นี้เป็นแอปพลิเคชันเว็บที่ใช้ Flask ในการสร้างเว็บแอปพลิเคชันเพื่อการรับข้อมูลภาพและการทำนายภาพตามภาษามือของคน โดยใช้โมเดลที่ได้ถูกฝึกสอนล่วงหน้าเพื่อการทำนายภาษามือภาษาไทย โดยมีขั้นตอนการทำงานดังนี้:

การตรวจจับและแสดง Landmarks: โปรเจ็คนี้ใช้ Mediapipe เพื่อตรวจจับและแสดง landmarks ของมือและโครงสร้างร่างกาย (pose) บนภาพ. จากนั้นจะมีการวาดเส้นเชื่อมระหว่าง landmark ต่าง ๆ เพื่อแสดงการเชื่อมโยง.

การประมวลผลภาพและการทำนาย: ข้อมูล landmark จะถูกนำเข้าไปในโมเดลที่ถูกฝึกสอนล่วงหน้าเพื่อการทำนายภาษามือ. โมเดลจะทำนายว่าสัญลักษณ์ภาษามือที่ปรากฏในภาพนั้นคืออะไรและความน่าจะเป็นของการทำนายด้วย.

การแสดงผล: การทำนายและสัญลักษณ์ภาษามือที่ตรวจจับได้จะถูกแสดงผลบนหน้าเว็บ พร้อมกับภาพที่มีการวาดเส้นเชื่อมระหว่าง landmark และสัญลักษณ์ที่ถูกทำนาย.

การรับข้อมูล: โปรเจ็คนี้รองรับการรับภาพทั้งจากการอัปโหลดไฟล์ภาพและการใช้กล้องเว็บ. ผู้ใช้สามารถอัปโหลดภาพที่ต้องการทำนายหรือใช้กล้องเว็บเพื่อถ่ายภาพและทำนายสดผ่านทางเว็บ.

การคำนวณความน่าจะเป็น: โมเดลจะคำนวณความน่าจะเป็นของการทำนายที่ได้ เพื่อแสดงความเชื่อมั่นในการทำนายของแต่ละสัญลักษณ์ภาษามือ.

การแสดงผลร่วมกับการบันทึกรูปภาพ: หลังจากที่ทำนายเสร็จสิ้น รูปภาพที่มีการวาดเส้นและสัญลักษณ์ที่ทำนายจะถูกบันทึกเก็บไว้ เพื่อให้ผู้ใช้สามารถดูผลการทำนายเป็นภาพได้.
